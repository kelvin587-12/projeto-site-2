# Projeto Web - Evolução para Interface Profissional

## Descrição do Projeto
Este projeto consiste na evolução da **Entrega I**, transformando a estrutura HTML inicial em uma interface visual **profissional, responsiva e acessível**, utilizando **CSS3** e boas práticas de design.

O foco principal foi criar uma experiência de usuário agradável, organizada e funcional, mantendo a compatibilidade com diferentes dispositivos e garantindo acessibilidade conforme padrões modernos.

---

## Evolução do Projeto

### Entrega I
- Estrutura HTML básica com conteúdo sem estilo.
- Layout simples, funcional apenas em desktop.
- Elementos sem responsividade ou efeitos visuais.

### Entrega II (atual)
- Aplicação de **CSS3** para estilização de toda a interface:
  - Layouts **responsivos** com Flexbox e Grid.
    - Cards de projetos com badges e tags, ajustáveis para diferentes telas.
  - Botões com **estados visuais** (`hover`, `focus`, `active`, `disabled`).
  - Formulários com **validação visual** e feedback instantâneo.
  - Componentes de feedback como **alerts e toasts**.
- Interfaces acessíveis:
  - Contraste adequado de cores.
  - Navegação por teclado.
  - Uso correto de elementos semânticos HTML.

---

## Tecnologias Utilizadas
- **HTML5**: Estrutura semântica e organizada.
- **CSS3**: Estilização avançada, responsividade e animações.
- **JavaScript (básico)**: Validação de formulários e feedback do usuário.

---

## Objetivos Alcançados
- Transformar um layout estático em uma **interface moderna e interativa**.
- Garantir **responsividade** para desktop, tablet e dispositivos móveis.
- Implementar **experiência de usuário agradável** com animações sutis e feedback visual.
- Aumentar a **acessibilidade**, tornando o site mais inclusivo.
